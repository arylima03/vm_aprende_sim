# FlutterFlow Setup

## Objetivo
Configurar a base do aplicativo no FlutterFlow conectada ao Firebase.

## Passos iniciais
1. Criar novo projeto no FlutterFlow.
2. Definir nome do app: VentEdu Sync.
3. Conectar projeto ao Firebase dev.
4. Ativar suporte Web/PWA.
5. Configurar tema global com paleta limpa, clinica e responsiva.

# Release Checklist PWA

## Antes do build
- Confirmar disclaimer visivel nas telas chave
- Confirmar conteudo revisado clinicamente
- Confirmar published apenas no conteudo liberado
- Confirmar links de navegacao

## Build
- Gerar build web
- Validar manifesto PWA
- Validar icones
- Validar nome curto e nome completo do app

## Testes
- Testar em desktop web
- Testar em iPad
- Testar em iPhone
- Testar Home, listas, detalhes, quiz e simulador
- Testar fallback baseline do simulador

## Seguranca
- Confirmar regras do Firestore aplicadas
- Confirmar AdminPage restrita
- Confirmar usuario comum sem acesso admin
- Confirmar ausencia de dados reais de pacientes

## Publicacao
- Publicar em ambiente dev ou piloto
- Registrar versao publicada
- Registrar data da release
- Registrar pendencias conhecidas

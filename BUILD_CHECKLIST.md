# Build Checklist

## Ordem recomendada
1. Criar projeto Firebase dev
2. Configurar Firestore e Authentication
3. Criar colecoes base
4. Importar seed inicial de conteudo
5. Criar shell do app no FlutterFlow
6. Implementar modulo Ajuste Inicial
7. Implementar modulo Assincronias
8. Implementar Simulador por estados
9. Implementar Casos e Quiz
10. Criar painel admin minimo
11. Testar PWA em iPhone, iPad e web
12. Publicar versao piloto

## Regras da v1
- Uso educacional e de simulacao
- Sem dados reais de pacientes
- Sem diagnostico ou prescricao automatica
- Sem integracao com ventilador real

## Projeto
1. Abrir FlutterFlow.
2. Criar novo projeto.
3. Nome do app: VentEdu Sync.
4. Escolher layout responsivo para web.

2. Ir em Firebase.
3. Conectar ao projeto Firebase dev correto.
4. Sincronizar Authentication.
5. Sincronizar Firestore.
